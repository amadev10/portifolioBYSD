# Portfólio Profissional – Engenheiro Eletrotécnico 

Este projeto consiste no desenvolvimento de um **site de portfólio profissional**
para um Engenheiro Eletrotécnico, com o objetivo de apresentar suas competências,
experiências, projetos e formação acadêmica de forma clara e moderna.

O site foi desenvolvido como um projeto web estático, focado em simplicidade,
boa apresentação visual e fácil navegação.

---

##  Objetivo do Projeto

- Criar uma presença online profissional
- Apresentar competências técnicas da área de Eletrotecnia
- Destacar projetos, experiências e áreas de atuação
- Facilitar o contato com recrutadores e clientes

---

##  Funcionalidades

- Página inicial com apresentação profissional
- Seção “Sobre”
- Seção de áreas de atuação
- Seção de projetos
- Seção de formação e experiência
- Seção de contato
- Layout responsivo

---

##  Tecnologias Utilizadas

- **HTML5** – Estrutura do site  
- **CSS3** – Estilização e layout responsivo  
- **JavaScript** – Interatividade e comportamento dinâmico
